# תכונות ושימוש

## מצלמה
- **גרירה בשטח ריק** = סיבוב · **גלגלת** = זום · **קליק-ימני + גרירה** = הזזה.
- **⬇ מבט על** — מצלמה אנכית מלמעלה (צפון למעלה).
- מצפן (N) בפינה שמאלית-תחתונה.

## מצב עריכה — 🔒 עדכן (בסרגל התחתון)
כברירת מחדל **נעול** (אי אפשר לשנות). לחיצה → **🔓 עדכן** (כתום = פתוח). רק במצב פתוח אפשר:
- **בחירה** — לחיצה על אובייקט (מגש/מעלית/צלחת/E1/קיר/פלורוסנט). מסגרת ירוקה + מידע בסרגל.
- **הזזה** — גרירת האובייקט הנבחר במישור X/Y.
- **סיבוב** — `Q`/`E` (5°, עם Shift 45°), או כפתורי `⟲ ⟳`; `R` = הצמדה ל-90°.
- **מחיקה** — `Delete`/`Backspace` או 🗑 מחק.
- **בניית קיר** — כפתור **➕ קיר** → לחיצה על נקודת התחלה ואז נקודת סיום. `Esc` = יציאה. הקיר גם ניתן לבחירה/הזזה/מחיקה.

> **פלורוסנטים** גם ניתנים לבחירה/הזזה/סיבוב במצב עריכה — כמו כל אובייקט.

## תפריט תצוגה (ימין עליון) — toggles
| Toggle | ברירת מחדל | פעולה |
|---|---|---|
| שרטוט (DWG) | ⬜ כבוי | שכבת השרטוט על הרצפה |
| מספרי חניה | ✅ דלוק | תוויות המספרים (**+ סמן E1** מוסתר/מוצג יחד) |
| סימון מרכזים | ⬜ כבוי | נקודות מרכז (מגשים/צלחת/מעלית) |
| קווי עזר | ⬜ כבוי | ציר המעלית × כיוון הנסיעה + נקודת הצטלבות |
| חושך מוחלט | ✅ דלוק | כיבוי אור סביבתי — רק פלורוסנטים |
| פלורוסנטים | ✅ דלוק | גופי התאורה בתקרה (אור + פסים) |
| תקרת קרקע | ⬜ כבוי | תקרת בטון חצי-שקופה |
| רכבים | ✅ דלוק | הצגת/הסתרת הרכבים |
| רשת | ⬜ כבוי | רשת רצפה |

בסרגל התחתון גם: **🚗 הסתר רכבים** (מסונכרן עם ה-toggle).

## שמירה, אתחול, איפוס
- **💾 שמור** — כותב את הפריסה ל-`rotem_saved.json` (דרך `serve.py`). **גם קובע את מצב האתחול** (baseline).
  - בלי `serve.py`: נשמר ל-`localStorage` (נשאר ברענון, לא לקובץ).
- **↺ אתחל סימולציה** — מחזיר את כל הרכבים/אובייקטים ל**מצב האחרון ששמרת** (מהיר, בלי רענון). שימושי אחרי הרצת תרחיש.
- **איפוס** — מוחק את הפריסה השמורה וחוזר לשרטוט המקורי (עם רענון + אישור).

> **זרימת עבודה בטוחה:** סדר → 💾 שמור (במנוחה) → ▶ הרץ תרחיש → ↺ אתחל. **אל תשמור באמצע תרחיש** (זה משמר מצב מעורבב).

## תפריט זמני סימולציה (⚙, שמאל עליון)
- **מהירות נסיעה** (מ׳/ש', ברירת מחדל 2.5) — תנועת רכבים/מגשים.
- **סיבוב צלחת** (°/ש', 120) — מהירות הסיבוב על הטרנטייבל.
- **מהירות מעלית** (מ׳/ש', 1.0) — מהירות עליית המעלית.
- **מספר רכב להוצאה** + **▶ הפעל חילוץ** — הקלד מספר והרץ.

הערכים חיים (משפיעים על התרחיש הבא).

## תרחישים (▶ הדמיה / ▶ הפעל חילוץ)
- **בלוק 1** (2–11, 22): חילוץ בסגנון חידת-15 → יציאה דרך צלחת → מעלית.
- **בלוק 2** (12–20): חילוץ עם רכב 21 כעוזר → יציאה דרך E1 → צלחת → מעלית.
- **רכב במעבר** (למשל 1): נסיעה לצלחת → סיבוב 360° → חזרה (הדמיה).
