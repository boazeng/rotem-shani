## מה זה הפרויקט

סימולציית **חניון רובוטי תת-קרקעי חד-קומתי** לפרויקט "רותם שני", שנבנתה מתוך **תכנון DWG/DXF אמיתי**. המתקן מבוסס על מערכת **Parkolay** (AGV) — כל רכב יושב על **מגש רובוטי** שנע במישור X/Y ומסתובב על **צלחת סיבוב (turntable)**, ויוצא דרך **מעלית רכב**.

הסימולציה היא **קובץ HTML יחיד** (Three.js) שרץ בדפדפן, עם:
- שחזור מדויק של צורת הבור המצולעי, המגשים, הצלחת, המעלית ו-E1.
- **מודלי רכב תלת-ממדיים אמיתיים** (11 דגמים).
- **עריכה אינטראקטיבית** — הזזה/סיבוב/מחיקה של מגשים, קירות, מעלית, צלחת, פלורוסנטים.
- **תרחישי חילוץ** בסגנון חידת-15 מבלוק 1 ובלוק 2.
- תאורת מרתף (פלורוסנטים), שמירה/טעינה לקובץ.

## הרצה מהירה

```bash
cd "rotem shani/simulation"
python serve.py
# פתח בדפדפן:  http://localhost:8010/rotem-shani.html
```

פרטים מלאים ב-[SETUP.md](SETUP.md).
